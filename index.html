Elementos Semanticos

u:
- Muestra el texto subrayado
- Sus etiquetas son: u y /u (Ambas obligatorias)
Ejemplo: Hola. Buen dia
i:
- Muestra el texto marcado con un estilo en cursiva o italica
- Sus etiquetas son: i y /i (Ambas obligatorias)
Ejemplo: Hola. Buen dia
abbr
- Representa una abreviación o acrónimo
- Puede ampliar o describir la abreviatura
Ejemplo:

I do HTML

b
- Indica que el texto debe ser representado
- Sus etiquetas son: "b" y "/b" (Ambas obligatorias).
- Puede contener: Texto y/o cero o más elementos
Ejemplo: Hola. Buen dia
br:
- Produce un salto de línea en el texto
- Util para escribir un poema o una dirección
- No utilizar "br" para incrementar el espacio entre líneas de texto
Ejemplo: Mozilla Foundation
1981 Landings Drive
Building K
Mountain View, CA 94043-0801
USA
cite:
- Marca una referencia a una fuente, o el autor de un texto citado
- Sus etiquetas son: "cite" y "/cite" (ambas obligatorias)
Ejemplo: Galileo dijo: "... y sin embargo, se mueve."
strong
- El elemento strong es el apropiado para marcar con especial énfasis las partes más importantes de un texto
- Sus etiquetas son: "strong" y "/strong" (ambas obligatorias)
Ejemplo: El dinero es importante pero la salud lo es más
s
- Muestra el texto tachado con una linea horizontal
- Sus etiquetas son: "s" y "/s" (Ambas obligatorias)
Ejemplo: Hola. Buen dia.
sub
- Define un fragmento de texto que se debe mostrar en subindice
- Ejemplo: La fórmula química de agua: H2O
sup
- Define un fragmento de texto que se debe mostrar en subindice
Ejemplo: 2 2
span
Es un contenedor en línea. Sirve para aplicar estilo al texto o agrupar elementos en línea
Ejemplo:

Ejemplo de div y span

Esto es un párrafo dentro de un div, y esto un span dentro de un párrafo.

time
- Representa un periodo específico en el tiempo
- Puede representar uno de los siguientes contenido:
· Una hora en formato de 24 horas
· Una fecha precisa en el Calendario Gregoriano (con hora y zona horaria opcionales)
· Un periodo de tiempo válido
Ejemplo: El concierto empieza a las 20:00
code
- Es el apropiado para marcar el código de un programa.
- Sus etiquetas son: "code" y "/code" (ambas obligatorias)
- Ejemplo: #include main() { for(;;) { printf ("Hello World!\n"); } }
anchor
- Crea un enlace a otras páginas de internet, archivos o ubicaciones dentro de la misma página
Ejemplo: +49 157 0156
